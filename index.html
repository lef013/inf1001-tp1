Introduction

Un hyperlien permet de lier un document HTML avec une autre ressource du Web: image, programme, document HTML (voir http://www.w3schools.com/), section dans un document HTML,

  • Un hyperlien à 2 extrémités appelées ancres: la source: point de départ de l'hyperlien et la destination: point d'arrivée du lien.
  • La destination de l'hyperlien est donnée par son URL
  • Élément a (a pour ancre)

  1. Adressage des ressources

    1. Adresse de base

      2. Chaque adresse URL a une adresse de base : c’est la partie de l’URL excluant ce qui suit le dernier « / ». Par exemple : L’adresse de base de
      http://www.uqtr.ca/~touref/INF1001/index.html est http://www.uqtr.ca/~touref/INF1001/ Habituellement le navigateur considère que l’adresse de base par défaut est l’adresse de base
      du document. La balise dans l’entête du document permet donner une adresse de base par défaut différente de celle du document.

    2. URL relatif

      3. L'attribut href spécifie l'URL de la destination de l'hyperlien. Cet URL peut être

      • Absolu: pour identifier une ressource indépendamment du contexte. http://www.tuteurs.ens.fr/internet/web/html/bases.html
      • Relatif: pour identifier une ressource dépendamment du contexte. Ce contexte est donné par l'adresse de base.

      • Par exemple,
        si l'adresse de base est http://www.uqtr.ca/~touref/INF1001/documents/, alors l'URL relatif lien/lesport.jpg désigne l'URL absolu
        http://www.uqtr.ca/~touref/INF1001/documents/lien/lesport.jpg
